## Requirements

For building and running the application you need:

- Java 11
- Gradle 7.6

## Steps to Build & Run
- open your command prompt, then:

```bash
gradle bootRun
# or
./gradlew bootRun
```
- Or, use IntelliJ to build and run using play button
